# Spotify Clone 🎵

A responsive Spotify web player clone built using **HTML, CSS and JavaScript** as a college mini project.

---

## 📌 About the Project

This is a front-end clone of Spotify's web player. It replicates the main UI of Spotify including the sidebar, home page with music cards, and the music player at the bottom.

> **Note:** This is just a UI clone for learning purposes. There is no actual music playback or Spotify API integration. All songs and playlists shown are static.

---

## 🚀 Features

- Sidebar with navigation (Home, Search, Your Library)
- Responsive music card grid (Recently Played, Trending, Featured Charts, Podcasts)
- Hover effects on music cards (green play button appears)
- Bottom player bar with:
  - Play / Pause toggle
  - Progress bar (clickable to seek)
  - Volume control (clickable)
  - Like / Heart button toggle
  - Song and artist name updates when a card is clicked
- Mobile responsive with hamburger menu
- Progress bar auto-simulates movement when playing

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Grid, Media Queries)
- Vanilla JavaScript (no libraries or frameworks)
- Google Fonts (Montserrat)
